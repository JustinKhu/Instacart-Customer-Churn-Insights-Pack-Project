# Instacart-Customer-Churn-Insights-Pack-Project


## Project Overview 
This project uses the Instacart Online Grocery Shopping Dataset with the aim of creating a valuable insights pack to help drive business decisions. This dataset will be cleaned and analysed in order to uncover any potential trends or useful information. Subsequently, any useful information found will be used to form recommendations to answer/solve a business problem (found in insights pack).
